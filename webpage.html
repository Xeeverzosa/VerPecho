What are the possible career paths for Senior High ICT Students?

Here are the possible careers after graduating as an ICT Students

ICT is a field full of amazing works and activities and ICT is really relevant this modern era of technologies because ICT plays a crucial role in various aspects of modern society, making it highly relevant across multiple domains.

CHARACTER ARTIST - Illustrates characters animators uses to Animate, it can be 2D,3D etc.

ANIMATION PRODUCTION MANAGER - Basically handles the whole process of making an animated film from Pre-production to Post-production

SOFTWARE DEVELOPER - Uses different computer language (JAVA,Phyton, CSS, HTML etc.) to create a System.

CYBER SECURITY MAKER - Builds and maintains a security for a system or program

WEB DEVELOPER - Develops and maintains a website that functions for a certain needs
